# 👨‍💻 Portafolio Interactivo - Despliegue en Cloud Run (GCP)

[![Estado: Activo](https://img.shields.io/badge/Estado-Activo-brightgreen.svg)]()
[![Deploy: Google Cloud](https://img.shields.io/badge/Deploy-Google%20Cloud%20Run-blue.svg)](https://cv-sebastian-torres-815761035274.us-west2.run.app/)

Portafolio web interactivo diseñado para simular una terminal de comandos. Este proyecto demuestra mis habilidades híbridas entre el desarrollo frontend y la administración de infraestructura Cloud (DevOps).

## 🚀 Enlace en Vivo
Puedes ver el portafolio corriendo en la nube de Google aquí:
**[Ver Portafolio Interactivo](https://cv-sebastian-torres-815761035274.us-west2.run.app/)**

## 🛠️ Stack Tecnológico
- **Frontend:** React.js, CSS3 (Animaciones, diseño responsivo y UX).
- **Despliegue & DevOps:** Google Cloud Platform (GCP), Google Cloud Run, Contenedores.
- **Asistencia de Desarrollo:** Estructura inicial y maquetación acelerada mediante *Google AI Studio*.

## ⚙️ Arquitectura y Toma de Decisiones
El diseño lógico, la configuración de infraestructura y el despliegue en la nube fueron realizados de manera manual. Para la generación de la interfaz web en React, utilicé herramientas de IA Generativa como asistentes de código. Esto me permitió optimizar drásticamente los tiempos de desarrollo Frontend para poder enfocarme en la seguridad y eficiencia del despliegue en contenedores dentro de GCP.

## 💻 Instalación Local

Si deseas correr este proyecto en tu entorno local para revisar el código:

1. Clona este repositorio:
   \`\`\`bash
   git clone https://github.com/sebatmail/portfolio-interactivo-cloud.git
   \`\`\`
2. Instala las dependencias:
   \`\`\`bash
   npm install
   \`\`\`
3. Ejecuta el servidor de desarrollo:
   \`\`\`bash
   npm run dev
   \`\`\`
